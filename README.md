> A lightweight, fast note-taking web app built with React + TypeScript + Vite.

## 🚀 Quick start

### Install

```bash
pnpm install
```

### Run locally

```bash
pnpm dev
```

Open `http://localhost:5173`.

### Build

```bash
pnpm build
```

### Format&Lint (if configured)

```bash
pnpm lint
pnpm format
```

## ✨ Key features

- Create, edit, delete notes
- Local storage persistence
- Search & filter notes (future enhancement)
- Keyboard shortcuts
- Modular component structure

## 🗂️ Project structure
